# A/B Testing 实验方案生成器

用 AI 生成规范的 A/B Testing 实验说明书，适用于 Claude、ChatGPT、Gemini 等任意 AI 工具。

---

## 什么是这个工具？

撰写 A/B Testing 实验方案是一件繁琐且耗时的事。当你已经和 AI 有了完整的上下文，或者手头有现成的需求文档时，不需要再人工逐字填写实验说明——直接让 AI 帮你整理输出。

这个提示词帮你一次性生成结构完整、逻辑自洽的实验说明书，涵盖从实验假设到分组说明的全部内容。

---

## 输出结构

| Part | 内容 |
|------|------|
| **Part 1** | 实验假设（If / Then / Because） |
| **Part 2** | 影响变量 / 一句话概述 |
| **Part 3** | 核心变量关系（有利因素 / 不利因素） |
| **Part 4** | 预期每日用户 |
| **Part 5** | 实验受众 |
| **Part 6** | 分组说明（V0 / V1 / V2…） |
| **Part 7** | Metrics（Primary / Secondary） |
| **Part 8** | Trigger |

---

## 使用方法

### 用 Claude.ai 的同事
下载 `ab-testing.skill` 文件，在 Claude.ai 中上传安装。安装后直接说"帮我写一个 AB Testing"即可触发。

### 用其他 AI 工具的同事
下载 `ab-testing-prompt.md`，复制里面的提示词到你使用的 AI 工具即可。

支持三种输入方式：
- **直接描述场景**：「在支付页增加倒计时，帮我写实验方案」
- **引用对话内容**：「根据我们的聊天内容，整理出这个项目的 A/B Testing」
- **上传需求文档**：「根据我给你的需求文档，帮我写实验方案」
